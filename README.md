# Evolutionary Intelligence Swarm (演化智能群)

基于OpenClaw的自我进化智能体群落系统

## 项目简介

**Evolutionary Swarm** 是一个模拟生物进化机制的AI智能体群落系统。通过遗传算法、变异、自然选择等机制，让智能体能够自主演化、协作解决问题，并在演化过程中涌现出更高层次的智能。

核心思想：将进化论应用于AI，让智能体像生物一样"适者生存"，通过思维模式的演化找到最优解。

## 核心特性

### 🎯 5种思维模式

| 模式 | 名称 | 特点 | 适用场景 |
|------|------|------|----------|
| analytical | 分析思维 | 逻辑严密，分解问题 | 数学、编程 |
| creative | 创造性思维 | 联想类比，创新解法 | 艺术、创意 |
| critical | 批判性质疑 | 质疑假设，验证推理 | 逻辑判断 |
| systematic | 系统性思维 | 全局视角，关系分析 | 复杂系统 |
| intuitive | 直觉思维 | 快速感知，模式识别 | 紧急决策 |

### 🧠 任务驱动 + 自主思维

- **任务驱动**: 智能体根据分配的任务自主选择最优思维模式
- **自主思维**: 元认知(MetaThinker)能够反思自己的思考过程，自我改进
- **思维链**: 每一步思考都有置信度记录，支持回溯和评估

### 🔄 演化机制

1. **基因变异**: 思维模式、深度、反思倾向均可变异
2. **自然选择**: 适应度=解题成功率×思维质量
3. **代际传承**: 优秀思维模式传递给下一代
4. **涌现智能**: 多代演化后群体涌现出超越个体的智能

## 运行方法

### 环境要求

```bash
Python 3.8+
```

### 快速开始

```bash
# 进入项目目录
cd F:\evolution_swarm

# 运行演示
python evolution_swarm.py

# 带参数运行
python evolution_swarm.py --generations 10 --population 20

# 查看帮助
python evolution_swarm.py --help
```

### 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--generations` | 演化代数 | 10 |
| `--population` | 种群大小 | 10 |
| `--problems` | 每代解决问题数 | 3 |
| `--mutation-rate` | 变异率 | 0.3 |
| `--mode` | 指定思维模式 | random |
| `--demo` | 运行交互演示 | False |

### Python API

```python
from evolution_swarm import EvolutionarySwarm

# 创建系统
swarm = EvolutionarySwarm(
    population_size=10,
    mutation_rate=0.3
)

# 添加任务
swarm.add_task("math_1", "求解: 2x + 5 = 15")
swarm.add_task("logic_1", "如果A>B, B>C, 则A>C?")

# 运行一代
result = swarm.run_once()

# 获取状态
status = swarm.get_status()
print(f"成功率: {status['success_rate']:.1%}")
```
